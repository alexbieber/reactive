# REACTIVE

Guided App Spec → Expo (React Native) codegen. See [PLAN.md](PLAN.md) and [docs/EXECUTION_PHASES.md](docs/EXECUTION_PHASES.md).

## Quick start

```bash
npm install
npm run dev              # wizard at http://localhost:5173
npm run codegen -- docs/spec-schema/examples/habit-tracker.spec.json ./out/MyApp
npm run check:artifact -- ./out/MyApp
```
